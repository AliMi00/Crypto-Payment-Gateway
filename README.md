# Crypto-Payment-Gateway
Crypto payment gateway with .net core 5 and sql 

This is mostly the backend of the crypto payment gateway website. (It can be change easily from MVC to API) that can accept wallet address for our customer that their payment goes to their wallet directly and its support payment to our wallet and then payment transferred to their wallet.


  •	Customer wallet address method use different transaction scans to check and add their transactions 
  •	Our wallet payment method uses exchange APIs to add and exchange the value to usdt for transferring to user account.
